BombStar Documentation

As you start the game 'Bomb Start', you will see the button selections.

New Game

the '"New Game" button will get you to start the game, after letting you to choose the game mode you wish to play.

  • Standard Mode
    • - Playing a Standard Mode game. A player is given initial 60 seconds to eliminate the blocks and get highest score as possible to set a record. This mode is encouraged to play with strategy, using bombs and score multipliers as well as several ways of shuffling the blocks.
  • Survival Mode
    • - Playing a Survival Mode. A player is given initial 30 seconds to clear the level. Try to get to high level with high scores to set a record. Basic rules are applied the same, and this mode is also encouraged with strategized playings.
  • Endless Mode
    • - Playing a Endless Mode. With unlimited time given to play, a player can get to know how basic rules apply to the game play, as well as practice the strategy to clear other modes.

High Scores

the "High Scores" button will show you the sets of scores you achieved by playing the game, categorized in each game mode.

Settings

the "Settings" button will let you change the 'Color Scheme' of the blocks and texts in the game. If you do not like the colors of the blocks, or having a trouble recognizing blocks in the game due to specialties concerning color blindness, you can change the 'Color Scheme'. you have five different sets of colors to choose, and you can also change this color scheme anytime while playing the game as well.

About

the "About' button" will show you the credits and contributors of the game, as well as short explanation of 'How to Play'. It will briefly describe about the goal of this game, score points, and three different modes of the game.

Quit

the "Quit" button will have you close this game application.
