    Guide to Setting Up Project
  1. Download files from the BombStar.zip file at the following link. Link to Dropbox folder.
  2. Create a Qt project named BombStar with the following properties:
    • Arm v5
    • Android NDK Tool chain = 4.4.3
    • Ant v1.8.2
  3. Copy C++ files to the project directory.
  4. Create an AVD with the following specifications:
    • API 10
    • SD Card Size = 1000
    • Snapshot: Enabled
    • Skin: Built-in = WVGA800
    • Abstracted LCD Density = 240
    • Max VM Application Heap Size = 24
    • Device RAM size = 256
  5. Be sure the following files are included in the project directory:
    • Block.h / Block.cpp
    • Colors.h / Colors.h
    • highScores.h / highScores.cpp
    • mainWindow_UI.h / mainWindow_UI.cpp
    • myRectItem.h / myRectItem.cpp
    • Score.h / Score.cpp
    • ScoreFrame.h / ScoreFrame.cpp
    • main.cpp
  6. Build project.
  7. Run project.
  8. Play game as described in the Instructions tab in this reference manual.
  9. Read the Design document to understand how the game is created and why it's created that way.
  10. Refer to the Doxygen Appendix for a quick reference to the objects used in the code. Note that the Doxygen page is self-contained and does not link back to this main page.
